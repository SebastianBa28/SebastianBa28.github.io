CDS 232 — Planar Drone:
Dynamics & Control

Project Overview
Full nonlinear analysis and control design for a 3-DOF planar quadrotor. Derived equations of motion from first principles, designed an LQR trajectory-tracking controller via linearization, and extended the system with Control Barrier Functions (CBF) to guarantee real-time obstacle avoidance via quadratic programming. Implemented entirely in MATLAB.

System Model

The planar drone is modeled as a rigid body with two independent thrust inputs u1 and u2 separated by moment arm r. The six-dimensional state vector is [x, ẋ, y, ẏ, θ, θ̇]T — horizontal and vertical position and velocity, plus pitch angle and rate. The equations of motion follow directly from Newton-Euler rigid body dynamics:

\[\begin{aligned} m\ddot{x} &= -(u_1 + u_2)\sin\theta \\[6pt] m\ddot{y} &= (u_1 + u_2)\cos\theta - mg \\[6pt] I\ddot{\theta} &= r\,(u_1 - u_2) \end{aligned}\] \[\mathbf{x} = \bigl[\,x,\;\dot{x},\;y,\;\dot{y},\;\theta,\;\dot{\theta}\,\bigr]^\top \in \mathbb{R}^6\]

Trajectory Tracking Controller

A minimum-snap optimization generates a smooth nominal reference trajectory over a sequence of waypoints. The controller then tracks this reference by linearizing the nonlinear dynamics about the reference trajectory, yielding time-varying linear error dynamics that admit an LQR solution.

Linearization & LQR Design

Defining the tracking error e = x − xref, the error dynamics are:

\[\dot{e} = Ae + Bu_e\] \[A = \frac{\partial f}{\partial \mathbf{x}}\bigg|_{\mathbf{x}_\text{ref},\,u_\text{ref}} , \qquad B = \frac{\partial f}{\partial u}\bigg|_{\mathbf{x}_\text{ref},\,u_\text{ref}} \] \[K = \operatorname{lqr}(A,\,B,\,Q,\,R)\] \[u = u_\text{ref} - Ke\]

Safety-Critical Control: Barrier Functions

The LQR controller is extended with a Control Barrier Function (CBF) layer to guarantee obstacle avoidance in real time. At each timestep, a small quadratic program (QP) minimally modifies the nominal control input to satisfy a safety constraint — keeping the system inside the safe set C = {x : h(x) ≥ 0}.

Formulation

\[h(\mathbf{x}) = \|x_\text{pos} - x_\text{obs}\| - D_\text{obs} \geq 0\] \[h_e(\mathbf{x},\,v) = \dot{h}(\mathbf{x},\,v) + \gamma_1\,h(\mathbf{x})\] \[\dot{h}_e(\mathbf{x},\,v,\,a) \geq -\gamma_2\,h_e(\mathbf{x},\,v)\] \[u^* = \underset{u}{\operatorname{arg\,min}}\;\tfrac{1}{2}\|u - u_\text{nom}\|^2 \quad \text{subject to CBF constraint}\]

Avoidance Results

Three values of γ are compared. All successfully avoid the obstacle — larger γ weakens the safety margin but reduces trajectory deviation.

Control Effort Trade-off

Tighter tracking demands higher control effort during avoidance, producing oscillations. A looser tracking weight yields smoother avoidance at the cost of larger steady-state error.
