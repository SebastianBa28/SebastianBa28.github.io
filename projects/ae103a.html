Ae 103a — CubeSat ADCS

Project Overview
Developed an Attitude Determination and Control System (ADCS) for CubeSat satellites using Reaction Wheels. Derived system dynamics using quaternion kinematics. Linearized dynamics to make use of linear control theory. Designed a Kalman Filter for state estimation and an LQR Controller for pointing correction. Implemented the overall system in Simulink to test the robustness of the controller against unmodeled disturbances.
